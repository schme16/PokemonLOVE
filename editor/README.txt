Select Layers: 1,2,3 (Number keys above keyboard)
Tilde(~) = Tile Selector
Mouse Left-Click places the currently selected tile (Holding shift places a 3x3 grid of tiles with the mouse at the center)

F-keys select modes:
	F1 = Map Painting
	F2 = Player Painting
	F3 = Delete Mode
	
	F12 = View all placed tiles (highlights them as different solid colour per layer    Purple=1 - Blue=2 - Orange=3).
	
	
CTRL+F = Fill layer with currently selected Tile.

CTRL+Delete = Clear Entire layer.

CTRL+S Saves the current map
